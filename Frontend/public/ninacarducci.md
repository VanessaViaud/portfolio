# Nina Carducci

Nina Carducci, photographe professionnelle, dispose d'un site web portofolio déjà bien rempli mais qui a besoin d'être amélioré. Ma mission était de corriger les quelques bugs, améliorer la rapidité de chargement du site, travailler sur le SEO et l'accessibilité du site.

## Les défis du projet :

- Améliorer la rapidité de chargement du site pour une meilleure expérience utilisateur : commencer par compresser les photos de Nina : ici choix fait de les convertir au format AVIF, puis minifier les fichiers JS et CSS.
- Travailler sur le référencement naturel : modifier le fichier HTML de bout en bout comme ajouter un titre dans le <head>, une méta description, parfaire le choix des balises sémantiques, notamment les balises <h>, intégrer avec shema.org les micro-données locales, etc.
- Corriger les erreurs d'accessibilité du site web : après identifications des erreurs avec le plugin WAVE, bien ajouter un texte alternif aux images, des aria-label sur les liens, des labels aux éléments de formulaires, et corriger les couleurs pour un meilleur contraste.
- Corriger les différents bugs du code : ouvrir les outils de développement du navigateur pour inspecter le code en temps réel, ajouter des points d'arrêt et des console.log pour suivre l'exécution du code étape par étape et repérer le bug pour le corriger.

## Compétences utilisées :

- Lighthouse
- Schema.org
- Open Graph
- Wave
- DevTools
